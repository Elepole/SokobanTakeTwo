SokobanTakeTwo
==============

School project of a little Sokoban in Java. must run in cli, have an editor, and other not so fancy feature.

Comment m’utiliser.

Lancer le programme sans option, c’est à dire sans argument, le jeu démarre par défaut au niveau 1.

Lancer le programme avec « —-level » plus, le nom du niveau, le jeu démarre au niveau demandé pour l’utilisateur.

Lancer le programme avec la méthode « editor », le mode éditeur démarre.
